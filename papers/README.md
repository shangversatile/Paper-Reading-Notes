# Papers

`papers/` stores verified paper notes.

A note is created only after primary-source verification.

Notes are organized around research-learning value.

Each note should record why the paper matters for technical understanding,
critical questions, and future experiment design.

Optional research context may be recorded when it clarifies how the paper
connects to an active or likely future research direction.

Each note should include critical questions and reproduction value.

Use `templates/paper-reading-template.md` for new entries.

Do not add unsupported claims, copied abstracts, or unverified citation metadata.
