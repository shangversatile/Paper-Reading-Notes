# Papers

`papers/` stores verified paper notes.

A note is created only after primary-source verification.

Notes are not restricted to advisor papers.

Each note should record why the paper matters independently of advisor fit.

Advisor connection is optional.

Each note should include critical questions and reproduction value.

Use `templates/paper-reading-template.md` for new entries.

Do not add unsupported claims, copied abstracts, or unverified citation metadata.
